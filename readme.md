## LaTeX Table Editor ##

**Version 0.3**

This is the GitHub page of an upcoming Open Source WYSIWYG table editor that exports to multiples languages including LaTeX, Plain TeX, HTML and Markdown.

### Use

You can use it [here](http://www.latex-tables.com/).

### Dependencies

You need Bootstrap 3 and JQuery. See the online example.

### Formats supported

 - LaTeX
 - Plain TeX
 - Markdown
 - HTML
 - CSV
 - Text (coming soon)
 - ConTeXt (coming soon)
 - BBCode (coming soon)
 - MediaWiki (coming soon)

### Status

The **LaTeX Table Editor** is in beta. No API is stable for now.
